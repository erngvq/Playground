## Environments
- conda create --name playground python=3.12
- conda activate playground
- conda env list
- conda deactivate
- conda remove -n playground --all

## Installing Libraries
- pip install -r requirements.txt
- conda install scikit-learn pandas numpy scipy matplotlib seaborn yellowbrick
- conda install -c conda-forge jupyterlab

## m2-base Utilities
- conda install m2-base
